# feedback.md

Append-only log of what real people say — beta users, quiz takers, comments, churn notes, creators. The single most valuable input to iteration. Log it **verbatim**, dated, with the source and the theme. Never paraphrase away the real language; the words people use are the raw material for the voice and the positioning.

Format per entry:

```
### YYYY-MM-DD — [source: beta / quiz / comment / churn / creator / DM]
"verbatim quote"
Theme: <one or two words>
Note: <does this change anything? which domino / decision does it touch?>
```
